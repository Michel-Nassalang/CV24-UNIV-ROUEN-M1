Aide - Service REST CV24

URL Méthode Opération Retour
/ GET Affiche la page d’accueil Format HTML ou XHTML valide
/help GET Affiche la page contenant les informations d’aide Format HTML ou XHTML valide
/cv24/resume/xml GET Affiche la liste des CV stockés Flux XML
/cv24/resume GET Affiche la liste des CV stockés Page HTML (ou XHTML)
/cv24/xml?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est transmis en paramètre. Flux XML conforme au schéma xsd cv24, comprenant un seul CV.
/cv24/html?id=<id> GET Affiche le contenu complet du CV dont l’identifiant est <id> Page HTML (ou XHTML)
/cv24/insert POST Flux XML respectant la spécification cv24 à ajouter, conforme au schéma xsd. Le flux XML retourné contient les informations suivantes : - id → numéro d’identifiant attribué à la spécification soumise - status → INSERTED.
En cas d’échec de l’opération, les informations de statut seront retournées - status → ERROR - detail → INVALID | DUPLICATED.
/cv24/delete/<id> DELETE Valeur de l'identifiant <id> Suppression du CV dont l’identifiant est <id>
